### Installing
```
nvm use 16.10.0
npm i -g pnpm
pnpm i

pnpm build && pnpm start
```

### Env
```
NEXT_PUBLIC_URL_SERVER_GRAPHQL=http://localhost:3000/api/graphql
URL_API=https://dummyjson.com/users
```

### Running app
```
pnpm run dev
```
